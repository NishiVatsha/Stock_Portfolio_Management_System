# 📈 Stock Portfolio Management System

### Java-Based Portfolio Tracking and Investment Management Application

A desktop-based application developed using **Java and Swing** that enables users to manage stock investments, monitor portfolio performance, and calculate profit/loss through an interactive graphical user interface.

## Overview

The Stock Portfolio Management System is designed to simplify the process of tracking and managing stock investments. The application provides a centralized platform where users can maintain their portfolio, view stock holdings, and analyze investment performance efficiently.

The project demonstrates the application of **Object-Oriented Programming (OOP)** principles, GUI development using Java Swing, and software design concepts to build a scalable and user-friendly system.

## Problem Statement

Investors often find it challenging to manually track multiple stock investments and evaluate portfolio performance. Without a structured system, monitoring holdings, calculating profits, and organizing investment records can become time-consuming and error-prone.

This project addresses these challenges by providing a simple and intuitive platform for portfolio management.

## Application Architecture

The system follows an object-oriented architecture:

* GUI Layer: Handles user interactions through Swing components.
* Business Logic Layer: Processes portfolio operations and calculations.
* Data Management Layer: Stores and organizes stock information.
* Analysis Layer: Computes investment performance and profit/loss metrics.
