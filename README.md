Stock Portfolio Management System
Overview

The Stock Portfolio Management System is a Java-based desktop application developed using NetBeans. It enables users to manage and track their stock investments through an intuitive graphical user interface. The system helps users maintain a portfolio, monitor stock holdings, and calculate investment performance efficiently.

Problem Statement

Managing stock investments manually can be time-consuming and error-prone. Investors need a centralized platform to track their holdings, monitor portfolio performance, and evaluate profit or loss. This project provides a simple and organized solution for portfolio management.

Features
Add stocks to portfolio
View stock holdings
Track portfolio performance
Calculate profit and loss
User-friendly graphical interface
Efficient data management
Modular and scalable design using OOP principles
Technologies Used
Java
Java Swing (GUI)

System Architecture

The application follows an object-oriented design where:

The GUI handles user interactions.
Portfolio management modules process stock data.
Business logic calculates portfolio statistics and profit/loss.
Data structures store and organize stock information.
